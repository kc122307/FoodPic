# FoodPic Calculator

Scan food with your camera to get nutritional information instantaneously.

## Features

- **Live Camera Detection**: Real-time AI detection of food items using your camera.
- **Image Upload**: Upload photos of your meals for detailed nutritional analysis.
- **Daily Calorie Tracker**: Track your daily caloric intake against your goals.
- **Portion Control**: Adjust portion sizes to get more accurate calorie counts.
- **AI Coach**: Get insights and recommendations based on your eating habits.
- **BMI Display**: Keep track of your Body Mass Index as you progress.
- **Exercise Recommendations**: Receive exercise suggestions based on your caloric intake.

## Project info

Scan food with your camera to get nutritional information. Built with modern web technologies for a smooth, app-like experience.

## Getting Started

Follow these steps to run the project locally:

1. **Clone the repository**
   ```sh
   git clone <YOUR_GIT_URL>
   ```

2. **Navigate to the project directory**
   ```sh
   cd foodpic-calculator
   ```

3. **Install dependencies**
   ```sh
   npm install
   ```

4. **Set up environment variables**
   Create a `.env` file based on the provided configuration (Supabase keys, etc.).

5. **Start the development server**
   ```sh
   npm run dev
   ```

## Technologies Used

- **Frontend**: React, Vite, TypeScript
- **Styling**: Tailwind CSS, shadcn/ui
- **Animations**: Framer Motion, GSAP
- **AI/ML**: TensorFlow.js, Gemini API
- **Backend**: Supabase
- **PWA**: vite-plugin-pwa

## Deployment

This project can be deployed on any modern static hosting provider like Netlify, Vercel, or GitHub Pages.

1. Build the project:
   ```sh
   npm run build
   ```
2. Deploy the `dist` folder to your chosen provider.
